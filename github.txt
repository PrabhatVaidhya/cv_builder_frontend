Short Description (for GitHub header):
-----------------------------------
An AI-powered CV tailoring platform that automatically optimizes resumes for specific job descriptions using advanced NLP and cosine similarity matching.


Detailed Description (for README):
-----------------------------------
AI CV Optimizer is a full-stack MERN application that revolutionizes the job application process by leveraging artificial intelligence to tailor CVs for specific job postings. The platform uses Groq's Llama 3.3 70B model to intelligently parse user profiles, match skills with job requirements, and generate professionally formatted, ATS-friendly CVs.

Key Features:
• AI-Powered CV Tailoring: Automatically customizes CVs based on job descriptions
• Intelligent Matching: Cosine similarity scoring (50-100%) to show relevance
• Profile Management: Edit and update profiles without losing historical data
• Formatted Preview: View CV exactly as it appears in PDF before downloading
• Professional PDF Generation: Clean, ATS-friendly document formatting
• Secure Authentication: JWT-based user authentication with encrypted passwords
• Dual Storage: MongoDB with in-memory fallback for reliability


Tech Stack:
-----------------------------------
Frontend:
- React 18.2.0 with Vite
- Modern UI with responsive design
- Real-time CV preview

Backend:
- Node.js/Express 4.18.2
- MongoDB with Mongoose ODM
- Groq AI API (Llama 3.3 70B - 500+ tokens/second)
- PDFKit for document generation

AI Integration:
- Natural Language Processing for profile parsing
- Cosine similarity algorithm for CV-job matching
- Context-aware content generation
- Temperature-controlled outputs (0.3-0.5)
